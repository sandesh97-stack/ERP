# 🎓 ERP COURSE REGISTRATION SYSTEM

A Python-based console application to manage student course registration and academic analytics.

# 🔗 Project Description

This ERP system simplifies academic workflows by integrating student registration, course management, and analytics into a single platform. It reduces manual effort and provides meaningful insights using data-driven techniques.

# 📌 Project Overview

This project was developed as part of our college coursework to demonstrate the practical implementation of Object-Oriented Programming (OOP), file handling, and data analysis in a real-world ERP scenario.

The system allows efficient management of students and courses while providing analytical insights such as credit statistics and department-wise distribution.

⚙️ Technologies Used:
- Python
- NumPy
- Pandas
- Matplotlib
  
# 🧠 Core Concepts Implemented
- Object-Oriented Programming (Classes, Inheritance)
- Modular Programming
- File Handling (Data Persistence)
- Data Analysis & Visualization
  
# 📂 Modules / Files
- main.py → Menu-driven interface
- models.py → Class definitions (User, Student, Course, Admin)
- operations.py → Core functionalities (registration, viewing, etc.)
- analytics.py → Data analysis and visualization
- file_handler.py → Data storage and retrieval
  
 # 🚀 Features
- 👨‍🎓 Student Registration
- 📚 Course Viewing
- 📝 Course Registration
- 📋 View Registered Courses
- 📊 Credit Analysis (Min, Max, Average)
- 📈 Department-wise Visualization (Bar Graph)
- 💾 Data Persistence using Files
  
# 📊 Functional Highlights
- Uses NumPy for statistical analysis of course credits
- Uses Pandas & Matplotlib for department visualization
- Implements inheritance (Student and Admin from User)
- Stores data in .txt files for persistence
